## 💙 TypeScript 입문

- 타입스크립트란 ? 자바스크립트의 확장된 언어라고 볼 수 있다. 타입스크립트는 자바스크립트와 다르게 브라우저에서 실행하기 위해 파일을 한번 변환해주어야 합니다. 이 변환 과정을 우리는 **컴파일** 이라고 부릅니다.
- 타입스크립트의 장점 : 에러의 사전 방지, 코드 가이드 및 자동 완성 (개발 생산성 향상)
