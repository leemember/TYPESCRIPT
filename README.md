## 💜 타입스크립트란 ?

> 타입스크립트는 자바스크립트에 타입을 부여한 언어다. 자바스크립트의 확장된 언어라고 볼 수 있다. 타입스크립트는 자바스크립트와 다르게 브라우저에서 실행하기 위해 파일을 변환해주어야 한다. 이 변환 과정을 <b>컴파일</b>이라 한다.
